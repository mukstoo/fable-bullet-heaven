# GRAVEHORDE

A dark-fantasy **bullet-heaven / survivors-like** for the browser. Survive the rising horde for 12 minutes and slay the Reaper.

Built one-shot by Claude (Fable 5) with Phaser 3 + TypeScript + Vite.

> Full README (how to play, architecture, credits) lands at the end of the build — see CREDITS.md for asset licensing.

## Dev

```bash
npm install
npm run dev      # local dev server
npm run build    # typecheck + production build to dist/
```
